## coding-agent-memory-evals

[coding-agent-memory-evals](https://github.com/CorrectRoadH/coding-agent-memory-evals) 对比 coding agent 的记忆机制对开发任务成功率的影响：同一批任务、同一个模型，只换 agent 是否带 [Tape](https://tape.systems/) 记忆。

- 在线报告：[niceeval.com/showcase/memory](https://niceeval.com/showcase/memory)
